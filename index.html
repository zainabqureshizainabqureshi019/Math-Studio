What is a Quadratic Equation?

A quadratic equation is a polynomial equation of degree 2. The standard form is:

ax² + bx + c = 0

where a, b, and c are constants, and a ≠ 0. If a = 0, the equation becomes linear, not quadratic.

Why "Quadratic"?

The word "quadratic" comes from the Latin word "quadratus", which means "square". This is because the variable x is squared (x²).

Real-Life Applications

  • Physics: Projectile motion – the path of a thrown ball follows a quadratic equation.
  • Engineering: Designing bridges and arches.
  • Finance: Calculating profit, loss, and break-even points.
  • Biology: Modeling population growth.

Key Components

  • a: Coefficient of x² – determines the shape of the parabola.
  • b: Coefficient of x – affects the position of the vertex.
  • c: Constant term – where the parabola crosses the y-axis.

How to Solve 2x² − 5x + 2 = 0 (Full Guide)

Let's solve the equation 2x² − 5x + 2 = 0 step by step using the quadratic formula.

Step 1: Identify a, b, and c

Compare with the standard form ax² + bx + c = 0:

  • a = 2
  • b = -5
  • c = 2

Step 2: Write the Quadratic Formula

x = (-b ± √(b² − 4ac)) / 2a

Step 3: Calculate the Discriminant

D = b² − 4ac = (-5)² − 4(2)(2) = 25 − 16 = 9

Since D > 0, there will be two distinct real roots.

Step 4: Plug into the Formula

x = (-(-5) ± √9) / 2(2) = (5 ± 3) / 4

Step 5: Find the Two Roots

  • x₁ = (5 + 3) / 4 = 8/4 = 2
  • x₂ = (5 − 3) / 4 = 2/4 = 0.5

Final Answer

The solutions are x = 2 and x = 0.5.

✅ You can verify by plugging these values back into the original equation.

5 Most Common Quadratic Equation Mistakes

Mistake #1: Incorrectly Identifying a, b, and c

Example: For 2x² + 3x = 5, students forget to move 5 to the other side.

Fix: Always write in standard form: 2x² + 3x − 5 = 0. Now a=2, b=3, c=-5.

Mistake #2: Forgetting the ± Sign

Example: Only writing x = (-b + √D)/2a and forgetting the negative.

Fix: Always write both: x₁ = (-b + √D)/2a and x₂ = (-b − √D)/2a.

Mistake #3: Misplacing Signs

Example: For x² − 4x + 4 = 0, some write a=1, b=4, c=4 instead of b=-4.

Fix: The sign of b is the sign that appears with x. Here b = -4.

Mistake #4: Incorrectly Simplifying

Example: (5 ± 3)/4 – some simplify to 2 ± 3/4 incorrectly.

Fix: Treat the numerator as a single expression: (5+3)/4 and (5-3)/4.

Mistake #5: Not Checking the Discriminant First

Example: Proceeding with the formula when D is negative.

Fix: Always calculate D first. If D < 0, write "No real roots" and stop.

Quadratic Formula vs Factoring: Which to Use?

Both methods solve quadratic equations, but which one should you use?

Quadratic Formula

  • Pros: Works for every quadratic equation. No guesswork.
  • Cons: More time-consuming and requires careful arithmetic.
  • Best for: Complex equations or when factoring is not obvious.
x = (-b ± √(b² − 4ac)) / 2a

Factoring

  • Pros: Much faster and simpler when it works.
  • Cons: Only works when the equation can be factored (not always possible).
  • Best for: Simple equations with integer roots.

When to Use Each

  • Use Factoring for: x² − 5x + 6 = 0 → (x − 2)(x − 3) = 0 → x = 2, 3
  • Use Quadratic Formula for: 2x² + 3x − 1 = 0 → Factoring is not obvious → use formula.

Rule of Thumb

  • If a, b, and c are small integers, try factoring first.
  • If factoring doesn't work quickly, use the quadratic formula.
  • If you're not sure, the quadratic formula always works!

Understanding the Discriminant (D = b² − 4ac)

The discriminant is the part under the square root in the quadratic formula.

D = b² − 4ac

What Does the Discriminant Tell Us?

The discriminant tells us about the nature of the roots without actually solving the equation.

Three Cases

Case 1: D > 0 (Positive)

The equation has two distinct real roots. The parabola crosses the x-axis at two points.

Example: x² − 4x + 3 = 0 → D = 16 − 12 = 4 → x = 3, 1

Case 2: D = 0 (Zero)

The equation has one real double root. The parabola touches the x-axis at exactly one point (the vertex).

Example: x² − 4x + 4 = 0 → D = 16 − 16 = 0 → x = 2 (double root)

Case 3: D < 0 (Negative)

The equation has no real roots. The parabola never crosses the x-axis.

Example: x² + 2x + 5 = 0 → D = 4 − 20 = -16 → No real roots

Quick Reference Table

Discriminant (D) Nature of Roots Graph
D > 0 Two distinct real roots Parabola crosses X-axis twice
D = 0 One double real root Parabola touches X-axis once
D < 0 No real roots Parabola never crosses X-axis

Why Does This Matter?

Checking the discriminant first saves you time. If D < 0, you can stop immediately and say "No real roots" without doing further calculation.
